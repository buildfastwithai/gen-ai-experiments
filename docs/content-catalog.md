# Content Catalog

This catalog standardizes discovery across projects and notebooks.

## Columns

- `title`
- `type` (app, notebook, workshop, roadmap)
- `category` (cookbooks, agents, rag, fine-tuning, experiments, etc.)
- `framework`
- `provider`
- `difficulty`
- `status`
- `path`

## Source of Truth

- Machine-readable catalog: `docs/content-index.csv`

## Maintenance

- Update the catalog whenever adding, removing, or moving content.
- Keep paths synchronized with the filesystem.
- Prefer stable tags for `category`, `difficulty`, and `status`.
