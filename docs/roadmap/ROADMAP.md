# GenAI Learning Roadmap

This roadmap turns the repository into a step-by-step curriculum.

Learn by building. At each level, complete small tasks before moving forward.

6. [Level 5 - Advanced GenAI](level-5.md)
7. [Level 6 - Specialization Paths](level-6.md)

## How To Use This Roadmap

1. Start at your current level.
2. Pick 1-2 projects listed in that level file.
3. Complete checkpoint tasks.
4. Write short notes on what worked and what failed.
5. Move to the next level.
