# Curriculum Checkpoints

Use this file to track progress while moving through the roadmap.

## Level-Based Checkpoints

## Level 0

- [ ] Python and environment setup complete
- [ ] One notebook executed without errors
- [ ] API keys configured with `.env`

## Level 1

- [ ] Prompt comparison task complete
- [ ] Structured output example complete
- [ ] Two-model comparison notes complete

## Level 2

- [ ] One simple app built and run locally
- [ ] One generator app built and run locally
- [ ] Basic input validation added

## Level 3

- [ ] One RAG app with your own documents
- [ ] One agent with at least one tool
- [ ] One multi-agent workflow demo

## Level 4

- [ ] Evaluation checklist created
- [ ] Regression test prompts defined
- [ ] Logging/tracing added to one app

## Level 5

- [ ] One multimodal workflow completed
- [ ] One MCP workflow completed
- [ ] One performance/cost report created

## Level 6

- [ ] One specialization path selected
- [ ] Two capstones completed
- [ ] Portfolio write-up published

## Portfolio Readiness Check

- [ ] At least 5 projects completed across multiple levels
- [ ] At least 2 projects include evaluation notes
- [ ] At least 1 project includes production-style monitoring/testing
- [ ] Capstones documented with problem, solution, tradeoffs, and results
