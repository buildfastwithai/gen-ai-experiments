# Naming Conventions

## Folders

- Use lowercase kebab-case.
- No spaces.
- No emoji characters.
- No temporary suffixes like `(1)`.

## Notebooks

Pattern:

```text
<topic>-<tool-or-model>-<purpose>.ipynb
```

Examples:

- `agents-langgraph-supervisor.ipynb`
- `rag-claude-from-scratch.ipynb`
- `openai-gpt-5-4-mini-cookbook.ipynb`

## README Files

- Always use `README.md` exactly.

## General

- Keep names descriptive but short.
- Prefer consistency over creativity in path naming.
