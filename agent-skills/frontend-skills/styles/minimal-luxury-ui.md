# Minimal Luxury UI

Design and implement understated, elegant interfaces where precision and restraint create a premium impression.

## Use When
- A user wants a high-end minimal website.
- Brand positioning is premium, exclusive, or boutique.
- The design should feel expensive without visual noise.

## Inputs To Collect
- Brand sector (fashion, hotel, skincare, jewelry, architecture).
- Emotional tone (quiet, bold, timeless, contemporary).
- Content volume and conversion priorities.
- Preferred visual references and constraints.

## Workflow
1. Define a strict reduction strategy: what gets removed.
2. Select premium typography pairings and whitespace system.
3. Build composition with strong alignment and sparse accents.
4. Add subtle interaction details (opacity, reveal, micro-lift).
5. Audit spacing consistency and contrast precision.

## Advanced Guidelines & Deep Dive
### Precision Fundamentals
- **Whitespace as Material**: Treat negative space as a visual asset equal to typography.
- **Type Discipline**: Use a narrow set of sizes/weights with strong consistency.
- **Tone Control**: Limit accents to one or two strategic moments.

### Interaction Language
- **Low-Amplitude Motion**: Gentle reveals and fades preserve calm tone.
- **Micro-Feedback**: Hover/focus states should feel precise, not theatrical.

### Anti-Patterns to Avoid
- **Pseudo-Luxury Cliches**: Random gold gradients and oversized sparkle effects.
- **Overcrowded Layouts**: Dense blocks undermine premium perception.
- **Weak Contrast**: Minimal style still requires clear readability.
