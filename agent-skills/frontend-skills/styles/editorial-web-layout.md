# Editorial Web Layout

Design and implement narrative, typography-led interfaces inspired by print editorials and modern digital magazines.

## Use When
- A user wants an editorial or magazine-like website.
- Content hierarchy and storytelling are primary goals.
- The brand needs cultural, premium, or art-direction-heavy presence.

## Inputs To Collect
- Content format (long-form, portfolio, campaign, publication).
- Voice and tone (serious, avant-garde, minimalist, poetic).
- Asset mix (photos, pull quotes, charts, side notes).
- Device priorities and reading accessibility expectations.

## Workflow
1. Define story flow and reading cadence from hero to footer.
2. Create typographic system (display serif + functional sans pairings).
3. Build modular grid with asymmetry and negative space.
4. Implement article components: pull quotes, sidebars, captions, callouts.
5. Tune responsive behavior to preserve reading comfort.

## Advanced Guidelines & Deep Dive
### Editorial Mechanics
- **Hierarchy by Type**: Lead with typography, not decoration. Headline, deck, body, and metadata must be unmistakable.
- **Grid Tension**: Controlled asymmetry creates energy; avoid collapsing everything into centered cards.
- **Image Choreography**: Alternate full-bleed visuals with tight text blocks to manage narrative rhythm.

### Content Components
- **Scannable Anchors**: Use subheads, highlights, and data callouts to support non-linear readers.
- **Context Layers**: Add bylines, timestamps, references, and side notes with quiet styling.

### Anti-Patterns to Avoid
- **Template Uniformity**: Repeating identical sections kills editorial character.
- **Over-Animation**: Excess motion disrupts long-form reading.
- **Tiny Body Text**: Stylish typography must remain comfortably readable on mobile.
