# Level 4 - Production Systems

## What You Learn

- Testing and evaluating model behavior
- Observability, monitoring, and reliability practices
- Designing app architecture for real usage

## Concepts

- Prompt and response evaluation
- Regression testing for LLM outputs
- Tracing and telemetry for agent/app flows
- Cost, latency, and quality tradeoffs

## Required Knowledge

- Level 3 complete
- Basic understanding of logs and metrics

## Checkpoint Tasks

- Add an evaluation checklist to one app
- Run tests on multiple prompts and compare failures
- Add basic tracing or logging to one app workflow
- Write a rollback plan for prompt/version changes
