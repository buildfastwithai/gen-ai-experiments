# Level 2 - Simple GenAI Apps

## What You Learn

- Build simple user-facing GenAI apps
- Connect prompts to UI and workflows
- Handle basic errors and user inputs

## Concepts

- Streamlit app flow
- Request/response handling
- Prompt chaining
- Basic safety filters and guardrails

## Required Knowledge

- Level 1 complete
- Comfortable with simple app execution (`python` or `streamlit run`)

## Checkpoint Tasks

- Build one text-only assistant app
- Build one generator app (quiz, README, blog, or ad copy)
- Add input validation for one user field
- Write a short note: what breaks when prompt is ambiguous
